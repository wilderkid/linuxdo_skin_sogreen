## ✨ 核心特性

*   **现代化卡片布局**: 采用严格的扁平化卡片网格设计，替代原生表格布局，信息结构更清晰。
*   **极简色彩哲学**: 以纯白/深黑为基底，搭配柔和的青/粉色作为点缀，减少视觉干扰，让您专注于内容本身。
*   **精致的细节优化**:
    *   帖子列表、正文、侧边栏、标签等全方位视觉重塑。
    *   为帖子正文中的图片添加了柔和的右下角阴影，提升层次感。
    *   对英文和数字进行高亮处理，便于快速识别关键信息。
*   **强大的字体自定义**: 内置“花枝丸”风格字体栈，并提供便捷的油猴菜单，让您轻松更换为任何您喜欢的本地字体。
*   **深色模式适配**: 完美兼容论坛官方的深色模式，提供一致的视觉体验。
*   **轻量与高效**: 脚本经过优化，无冗余代码，不影响论坛加载速度。

## ✍️ 如何更换字体？

本脚本允许您通过油猴扩展菜单轻松自定义论坛的全局字体。

**操作步骤:**

1.  **打开油猴菜单**:
    *   在浏览器右上角找到并点击 **Tampermonkey (油猴)** 图标。
    *   在弹出的菜单中，您会看到脚本名称下方出现了两个新的选项。

    

2.  **设置自定义字体**:
    *   点击 **“设置自定义字体”** 选项。
    *   会弹出一个输入框，里面是当前正在使用的字体列表。
    *   您可以将其修改为您电脑上已安装的任何字体。
        *   **字体名称**: 请使用字体的英文名或中文名，并用双引号包裹，例如 `"Microsoft YaHei"`。
        *   **多个字体**: 您可以设置多个备用字体，用英文逗号 `,` 隔开。浏览器会从左到右依次尝试加载。
        *   **示例**: `"霞鹜文楷", "仓耳今楷", sans-serif`

3.  **恢复默认字体**:
    *   如果您想换回脚本内置的默认字体，只需点击 **“恢复默认字体”** 即可。
